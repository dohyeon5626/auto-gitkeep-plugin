This JetBrains plugin automatically creates a .gitkeep file in empty folders. You can either manually select a folder to add or remove the .gitkeep file, or use the automatic feature to have .gitkeep files added whenever a new folder or file is created. This ensures that empty folders are included in your Git repository without the need for extra manual steps.<br/>
